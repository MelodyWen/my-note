# linux装机笔记
### 1. 设置语言
设置英文的uft8
```sh
apt-get install -y language-pack-en-base
locale-gen en_US.UTF-8
LC_ALL="en_US.UTF-8"
LC_CTYPE="en_US.UTF-8"
LANG="en_US.UTF-8"
```
设置中文语言包
```sh
sudo apt-get install language-pack-zh
or
sudo locale-gen zh_CN.UTF-8
```
Set Timezone-
```
ln -sf /usr/share/zoneinfo/Asia/Shanghai /etc/localtime
```
